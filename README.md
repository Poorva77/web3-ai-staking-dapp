# Web3 AI Staking DApp

A full-stack decentralized application that combines **Blockchain**, **Artificial Intelligence**, and **Modern Web Development**.

## Features

* Staking Smart Contract using Solidity
* Frontend built with React.js
* Backend APIs using Node.js & Express.js
* MetaMask wallet integration
* Machine Learning mini-experiment
* Security analysis and report
* Deployment on SecureChain Mainnet (SCAI)

## Tech Stack

| Category   | Technologies                      |
| ---------- | --------------------------------- |
| Blockchain | Solidity, Remix IDE, SCAI Mainnet |
| Frontend   | React.js                          |
| Backend    | Node.js, Express.js               |
| Database   | MongoDB                           |
| Wallet     | MetaMask                          |
| AI/ML      | Python, Scikit-learn              |
| Deployment | Vercel                            |
